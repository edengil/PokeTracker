# Pokemon Tracker

## Running instructions

### Initialize the DB

1. run the file

   ```
   createDB.sql
   ```

   which creates the poke_tracker DB.

2. run the file

   ```
   create_tables.sql
   ```

   which creates 2 tables inside poke_tracker DB:

   - pokemons
   - trainers

3. run the file
   ```
   create_linking_table.sql
   ```
   which creates the table "pokemons_trainers" inside poke_tracker DB.
